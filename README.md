# DT-Clinic-Business-Analyst-Assignment
Clinic Operations &amp; Patient Communication System Design
This repository contains my submission for the DT Business Analyst assignment.

The assignment focuses on designing simple, practical systems for small clinics to:

- protect doctor time
- improve patient communication

It includes:
- Task 1: Daily Clinic Inventory Micro-Audit
- Task 2: Patient Care & Communication System
